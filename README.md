此仓库主要用于存储使用EuroScope创作模拟机文本的教程。
将提供模拟机文本（Sweatbox）从撰写的完整教程。


`main.py`为模拟机文本撰写程序，`/adf/RouteCheck.csv`为航路文件存放处，`/adf/STARSID.csv`为进离场航路存放处，`Gate.json`为各机场机位经纬度&头朝向存放处。

其中，`/adf/STARSID.csv`及`Gate.json`的格式均为原创。
